# SLIIT Eats - Canteen Management System

A comprehensive canteen management system for SLIIT university with loyalty program features.

## Features

- Browse canteens and their menus
- Shopping cart functionality
- Loyalty points system (1 point per Rs. 1000 spent)
- Rewards redemption (including 50% off next purchase)
- Blog and news updates from canteens
- User authentication
- Admin dashboards

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Loyalty Program

- Earn 1 point for every Rs. 1000 spent
- Redeem 10 points for 50% off your next purchase
- Other rewards available at individual canteens

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Radix UI components
- React Router
- React Query
